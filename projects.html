Reflexion

A 2D top-down game where the player must navigate their clone through increasingly difficult levels to reach the exit. Each level introduces new obstacles and game mechanics that test the player's logic, reflex, and spactial awareness.

View Source Code on GitHub

Features

  • Clone-based movement system with collision detection
  • 3 original levels with increasing difficulty
  • Background music and sound effects implemented using SDL_mixer
  • Shader-based visual effects with transitions and lighting

Tech Stack

  • C++: core game logic and architecture
  • OpenGL: rendering spritesheets and tilesets
  • SDL2/SDL_mixer: window management, input handling, audio playback
  • Custom game loop and level loading from array-based map definitions
